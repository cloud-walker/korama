# Vision

Korama is built looking forward, not backward. It targets modern runtimes and the current direction of the ecosystem, and deliberately declines to support legacy environments. We would rather adopt where the platform is heading — modern ESM, the latest React major, current Node — than carry the weight of where it has been. When a decision trades broad backwards compatibility for a simpler, more modern foundation, that trade-off is intentional and aligned with this vision.
